# Language

This game was made for Turbo Pascal 7.0, using the `kojakvga` (version 3.5) and
`smix` (version 1.25) units.

# Compiling

* Launch Turbo Pascal 7.0.
* Run `savefont.pas`.
* Compile `sponge.pas`.

# Running

* Now you can run `sponge.exe`.
* As for how the game works, see [sponge.md](sponge.md).
